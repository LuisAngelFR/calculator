# Calcularora

## Descripción
Esta es una calculadora muy básica desarrollada en Python para la escuela.

## Funcionalidades
- Suma
- Resta
- Multiplicación
- División

## Menú
```
1. Suma
2. Resta
3. Multiplicación
4. División
5. Suma avanzada
6. Salir
```

## Modo de uso
1. Clonar el repositorio
2. Abrir la terminal
3. Navegar a la carpeta del proyecto
4. Ejecutar el comando `python main.py` o `python3 main.py`
